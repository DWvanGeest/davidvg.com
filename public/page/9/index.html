60cycleCMS 2.1.0

Aug 16th, 2009 | Comments

Yep, another release :-). Since 2.0.0 I've fixed a few security vulnerabilities and bugs, and now 2.1.0 brings a better installation script that Does More For YouTM, including:

  • Creating the database from scratch, if needed
  • Automatically creating the .htaccess and .htpasswd files to protect the administrative area of the site, based on credentials chosen by the user
  • Attempting to create the necessary MySQL credentials file

The Dashboard now includes links to change the username and password as well. I've also set up a SourceForge site for the project, although there isn't terribly much on there for now.

Next on my TODO list is allowing visitors to leave comments.

Grab the latest zip from the project page or use anonymous SVN access. Enjoy!

60cycleCMS 2.0.0 released

Jul 25th, 2009 | Comments

I'm pleased to announce the release of 60cycleCMS 2.0.0, a major update to the open-source Content Management System powering this site.

New features and updates include:

  • Use MySQL to store everything, don't put entries in XML files anymore (that was a dumb idea)
  • New private "Dashboard" page, giving easy access to all site administration pages
  • Ability to edit and delete entries using the Dashboard web interface
  • Full support for HTML when publishing new entries
  • Ability to preview and edit new entries before publishing them
  • Time and timezone automatically detected when new entry is published
  • New installation script to automatically create MySQL table for entries (you still need to create the DB yourself)
  • Installation documentation

For anybody migrating from previous versions of 60cycleCMS, I've also written a script to import your old XML-stored entries to the new DB system.

I've been using 2.0.0 on this website for the last week or so, and so far I'm a big fan :-). Hopefully you are too, but if you run into any problems, shoot me an email.

Stuff I Couldn’t Find on the Internet: MichCon Edition

Jul 23rd, 2009 | Comments

I'm moving (again), but not back to Canada this time. My new apartment is in the area of Grand Rapids that Google Maps refers to as "South Hills"… not that I've ever heard it called that.

Anyway, long story short, I had to register for new gas service with DTE Energy, which of course involved a failed web form submission, a phone call, and finally two trips down to my "local" office. Why two trips you ask? Because nowhere on the Internet could I find a phone number for this location or its hours of operation. So, for the benefit of all Grand Rapidians, here's the goods:

At the time of this writing, DTE Energy, aka MichCon, could be found at 3538 Lake Eastbrook SE, Grand Rapids, and was open from 9 am to 4:30 pm, Monday to Friday.
