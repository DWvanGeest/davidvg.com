60cycle.net gets a spiffy menu

Mar 30th, 2009 | Comments

I just finished upgrading the left-hand menu to support expandable and collapsible sub-menus (click on "Source" to see it in action, you'll need Javascript enabled). The new menu uses jQuery and a customized version of this DynamicDrive script.

The motivation for this undertaking? The addition of a PHP Scripts page to the "Source" section (formerly "Projects"). So far in this section I've added some handy scripts that I use to dynamically generate the version field and download links on the 60cycleCMS page.
