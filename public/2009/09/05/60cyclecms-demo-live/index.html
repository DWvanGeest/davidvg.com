60cycleCMS demo live!

Sep 5th, 2009 | Comments

For anyone interested in trying 60cycleCMS before installing it, there's now a full-featured demo installation available on OpenSourceCMS. This does have the caveat that I do not control either the source code or the content of the demo, so if it's broken or someone insults your grandmother on it, it's not my fault.

So far though, I'm very impressed with OpenSourceCMS. It seems like a great resource for both users and developers of CMSes.
